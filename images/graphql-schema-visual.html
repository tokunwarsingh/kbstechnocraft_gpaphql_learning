What is a GraphQL Schema?

A GraphQL schema is the blueprint for your API that defines the structure, types, and relationships of your data. It serves as a contract between the client and server.

  • Defines available data types
  • Specifies allowed operations
  • Enables type safety & validation
  • Self-documenting API

Schema Components

Object Types

type Country {
  code: ID
  name: String
  capital: String
  currency: String
  phone: String
}

Query Type

type Query {
  countries: [Country]
  country(code: ID!): Country
}

Scalar Types

  • String - Text values
  • Int - Integer numbers
  • Float - Decimal numbers
  • Boolean - True/False
  • ID - Unique identifiers
